# Facial Emotion Detection Web Application

This project is a **Facial Emotion Recognition Web App** that detects a person's emotion from an image using a **retrained deep learning model** based on the **FER-2013 dataset**.  
The model was trained in **Google Colab**, exported as `emotion_model.h5`, and integrated into a **Flask-based web application**.
